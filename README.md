# AgroPest Control for Trading

Modern responsive website for an Egyptian agricultural input company built with Next.js and Tailwind CSS.

Editable site copy, navigation, card data, contact details, and image paths live in `lib/siteContent.ts`.

## Pages

- Home
- About
- Solutions
- Products
- Technical Library
- Partner With Us
- Contact

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
